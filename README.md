# 💾 SavingSync

**Control inteligente de tu ahorro y gastos personales**

Una aplicación web moderna y responsive para gestionar tus finanzas personales, registrar gastos diarios, analizar patrones de gasto mensual y hacer seguimiento de tu ahorro.

## ✨ Características

- 🔐 **Sistema de Login** - Crea tu cuenta con usuario y contraseña
- 📝 **Registro de Gastos** - Añade gastos con categorías, descripción y fecha
- 📊 **Análisis Mensual** - Ve tu sueldo, gastos, ahorro disponible y porcentaje ahorrado
- 💰 **Seguimiento de Ahorro** - Registra tu ahorro mensual y ve el total acumulado
- 🏷️ **Categorías Personalizables** - Crea tus propias categorías de gastos con emojis
- 📱 **Diseño Responsive** - Funciona perfectamente en móviles, tablets y desktop
- 🎨 **Interfaz Cyberpunk** - Tema neon oscuro moderno y atractivo
- 💾 **Almacenamiento Local** - Todos tus datos se guardan en tu dispositivo (sin servidor)

## 🚀 Inicio Rápido

### Opción 1: Uso Local (Recomendado)
1. Descarga el archivo `savingsync-login.html`
2. Abre el archivo en tu navegador (Chrome, Safari, Firefox, Edge)
3. ¡Listo! La app funciona sin conexión a internet

### Opción 2: En tu Móvil (iOS/Android)
1. Descarga `savingsync-login.html` 
2. Abre en Safari (iOS) o Chrome (Android)
3. Toca Compartir → "Añadir a pantalla de inicio"
4. Se instalará como app nativa

### Opción 3: En Servidor Web
1. Sube `savingsync-login.html` a tu hosting
2. Accede desde cualquier navegador

## 📋 Funcionalidades Detalladas

### 🔑 Autenticación
- Crear nueva cuenta con usuario y contraseña
- Iniciar sesión con credenciales
- Cambiar contraseña desde Configuración
- Cerrar sesión
- Datos privados por usuario

### 💳 Registro de Gastos
- Seleccionar categoría (con emojis)
- Escribir descripción del gasto
- Ingresar cantidad en euros
- Seleccionar fecha
- Ver últimos gastos registrados

### 📅 Vista de Hoy
- Total de gastos del día
- Lista detallada de gastos de hoy

### 📊 Análisis del Mes
- Sueldo mensual
- Gastos totales del mes
- Ahorro disponible
- Porcentaje de ahorro
- Análisis comparativo

### 💰 Gestión de Ahorro
- Ver ahorro total acumulado
- Añadir ahorro de cada mes
- Historial de ahorros mensuales

### ⚙️ Configuración
- Cambiar datos personales (sueldo, ahorro)
- Cambiar contraseña
- Crear categorías personalizadas
- Eliminar categorías
- Cerrar sesión
